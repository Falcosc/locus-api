What is it:

- in late 2019, main icon for Locus Map application change. It is recommended to use this new icon for new created or updated add-ons.

How to use it:

 - extract content of *ic_launcher.zip* file
 - copy it's content into own application *res* folder
 - modify all icons to match your own needs
 - set icon for you application in manifest.xml as *android:icon="@mipmap/ic_launcher_no1*
